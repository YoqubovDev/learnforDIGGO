Maxsus Teglar
Boshqa maxsus teglar
<title> Sahifa nomi
<meta> Metama'lumot
<link> Bog'lanish
<script> Skript
<hr> Chiziq
<br> Qator
src
Rasm yoki manbaning manziliga ishora qiladi
<img src="photo.jpg">
alt
Rasm ko'rinmasa, ko'rsatiladigan matn
<img alt="My Photo">
href
Havolaning manzili (yo'nalishi)
<a href="page.html">
placeholder
Input maydonda ko'rsatiladigan yordamchi matn
<input placeholder="Ism kiriting">
required
Input maydoni to'ldirilishi majburiy
<input required>
readonly
Inputni o'zgartirib bo'lmaydi, faqat ko'rish
<input readonly>
type
Input turini belgilaydi (text, email, number, etc.)
<input type="email">
value
Input elementining dastlabki qiymati
<input value="Default">
max
Raqamli inputning maksimal qiymati
<input type="number" max="100">
min
Raqamli inputning minimal qiymati
<input type="number" min="0">
action
Forma yuborilganda qayta ishlanish manzili
<form action="submit.php">
maxlength
Maksimal belgilar soni
<input maxlength="20">
minlength
Minimal belgilar soni
<input minlength="5">
id
Element uchun unikal identifikator
<div id="header">
class
CSS stilini qo'llash uchun sinfi
<div class="container">
colspan
Katakning nechtadir ustunni birlashtirishini belgilaydi
<td colspan="2">
rowspan
Katakning nechtadir qatorni birlashtirishini belgilaydi
<td rowspan="2">
scope
Sarlavha katagi (th) ustun yoki qatorga tegishli ekanligini belgilaydi
<th scope="col">
headers
Katakni bir yoki bir nechta sarlavha kataklari bilan bog'laydi
<td headers="header_id">
